### Ubuntu

```
rename 's/\.js/\.ts/' *
```

### CentOS

```
rename .js .ts *.js
```

### find & xargs

```
find -maxdepth 1 -name '*.js' | awk -F '.' '{print $2}' | xargs -i -t mv ./{}.js  ./{}.ts
```
